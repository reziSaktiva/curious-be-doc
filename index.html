getPosts

Description

query ini digunakan untuk mengambil beberapa postingan dengan kondisi tertentu, ada tiga kondisi yang sudah di siapkan dalam query getPosts.

  1. di curious app user dapat melihat beberapa konten terbaru (latest) atau popular. dalam kondisi seperti ini query dapat digunakan dengan cara mengisi variable dengan lat, lng, range (boleh tidak di isi), dan type. variable yang lainnya di lewat saja.
  2. di curous app pun user dapat melihat beberapa postingan di profile nya khususnya postingan yang sudah di post oleh user itu sendiri dan juga kita dapat melihat beberapa profile pengguna lain juga. untuk menggunakan query pada kondisi ini kita dapa mengisi variables hanya dengan username saja dan tidak perlu mengisi variable lain.
  3. curious app juga menyediakan fitur room, room adalah fitur yang di sediakan oleh curious app seperti grup hanya saja memiliki wilayah tertentu. pengguna hanya dapat mengakses room yang tersedia di wilayah mereka. untuk menggunakan getPosts untuk mendapatkan room post kita hanya perlu mengisi variables room.
Response

Returns a dataPost

Arguments
Name Description
lat - Float -6.881313224494528 (latitude)
lng - Float 107.59699743723579 (longitude)
range - Float 10 (angka satuan KM)
type - String Latest atau Popular
room - ID 5OW16cm76N6nrO0ZgGLm (id room)
userId - String id dari userData

Example

Query

  $lat: Float,
  $lng: Float,
  $range: Float,

  $room: ID,
  $userId: String

    range: $range,

    room: $room,
    userId: $userId
  ) {
    hasMore

    }

  }
}
